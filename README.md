# simple-stream-dump

Consume [simple-stream](https://github.com/eldargab/stream-simple)
throwing everything away.

## Example

```javascript
var dump = require('simple-stream-dump')
dump(stream)
```

## Installation

via npm

```
npm install simple-stream-dump
```

via component

```
component install eldargab/simple-stream-dump
```
